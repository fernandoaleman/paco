# Bundled icon attribution

paco's `applications/icons/` ships PNG icons used by the default
web-app `.desktop` entries created by `install/packaging/webapps.sh`.
Sources and license terms are tracked here so users redistributing
paco (forks, ISOs, etc.) can comply with each icon's terms.
